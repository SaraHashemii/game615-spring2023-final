# game615-spring2023-final
 Sara <br>

**Title:** Abandoned in the Snow<br>
**Game Objectives:** kill all snowmen and collect the carrots to survive.<br>

**Game controls:**<br>
W,A,S,D = move<br>
Shift = run<br>
mouse = camera<br>
left click = shoot<br>
